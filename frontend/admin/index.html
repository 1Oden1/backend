✏️ Saisir / modifier une note
ℹ️ Comment saisir une note ?

1. Allez dans Étudiants pour trouver l'ID de l'étudiant.

2. Allez dans Calendrier → Modules pour trouver l'ID de l'élément.

3. Saisissez la note ici. L'opération est un upsert — si une note existe déjà elle sera mise à jour.

Pour visualiser toutes les notes d'une promo, utilisez l'onglet 📊 Vue notes.
