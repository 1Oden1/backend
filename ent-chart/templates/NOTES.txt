⚠️  ÉTAPES POST-INSTALLATION

  1. Importer le realm Keycloak si pas auto-importé :
     kubectl create configmap keycloak-realm-config \
       --from-file=realm-export.json=ms-auth/keycloak/realm-export.json \
       -n {{ include "ent.namespace" . }}

  2. Vérifier le pull du modèle llama3 dans Ollama :
     kubectl logs -n {{ include "ent.namespace" . }} \
       $(kubectl get pod -n {{ include "ent.namespace" . }} -l app.kubernetes.io/name=ollama -o name) \
       -c pull-model

  3. Surveiller le scaling automatique :
     watch kubectl get hpa -n {{ include "ent.namespace" . }}

  4. DNS : pointer {{ .Values.global.domain }} vers l'IP de l'Ingress Controller :
     kubectl get svc -n ingress-nginx ingress-nginx-controller
